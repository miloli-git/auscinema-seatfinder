AusCinema Seat Finder - UI exploration

Seven whole-page directions for the same product: find the cinema seat actually worth sitting in. Every mock shares the same data, so the only variable is design. Several are grounded in design systems extracted from live sites; the hybrid renders a real auditorium.

Same data across all mocks (Toy Story 5 / Event George St / 6 sessions) grounded built from a system extracted off a live site real layout renders a real captured auditorium, not a synthetic grid

Method: design systems extracted with Playwright into extractions/*-DESIGN.md; real auditorium geometry pulled through the project's own chain adapters and scorer into data/real-layouts.json. See README for the write-up. Every mock is a single self-contained file (no external fonts, images, or scripts).
