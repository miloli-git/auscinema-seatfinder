AusCinema Seat Finder - UI exploration

Seven whole-page directions for the same product: find the cinema seat actually worth sitting in. Same data across every mock, so the only variable is design. Drag the deck, or use the arrows.

Same data (Toy Story 5 / Event George St / 6 sessions) grounded system extracted off a live site real layout renders a real captured auditorium View as grid ->

Method: design systems extracted with Playwright into extractions/*-DESIGN.md; real auditorium geometry pulled through the project's own chain adapters and scorer into data/real-layouts.json. See README for the write-up. Each mock is a single self-contained file. Carousel technique: Stripe case-study flipbook (mandatory scroll-snap + edge-peek + container-query sizing + drag-momentum).
